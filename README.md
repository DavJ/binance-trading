# binance-trading



Install binance RPM on Ubuntu: 
   wget https://ftp.binance.com/electron-desktop/linux/production/binance-x86_64-linux.rpm
   sudo apt install alien
   sudo alien -i binance-x86_64-linux.rpm --scripts
